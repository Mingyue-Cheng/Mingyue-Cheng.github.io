Biography

I am an Associate Researcher at the School of Computer Science and Technology, USTC. I am also affiliated with the State Key Laboratory of Cognitive Intelligence at the side of USTC, working under the leadership of Prof. Enhong Chen and Prof. Qi Liu. Previously, I obtained my Ph.D. degree under the supervision of Prof. Qi Liu.

Research Interests

My research focuses on LLMs and Agentic AI, with applications in Structured Data Mining, Recommender Systems, and AI for Science.

  • 🤖 LLMs and Agentic AI: Focused on agent runtime construction — including memory, context management, and skill libraries — agentic RL across algorithms, training frameworks, and system infrastructure, and agent action behavior analysis covering action mechanisms, decision interpretability, and emergent tool-use patterns.
  • 📊 Structured Data Mining: Building foundation models and agentic frameworks for time series, spatio-temporal, and tabular data — covering forecasting, classification, anomaly detection, and universal data representation.
  • 🎯 Recommender Systems: Context-aware personalized recommendation via sequential modeling, one-class collaborative filtering, and LLM-driven reasoning for dynamic user preference understanding.
  • 🔬 AI for Science: Focused on structured scientific data mining — including scientific tables and time-series — and scientific literature mining, covering information extraction, knowledge reasoning, and agent-driven literature understanding.
Research collections: 🤖 Agentic AI · 📊 Time Series Analysis · 📋 Table Mining · 🔬 Scientific Literature Mining

Open Source

CrabShrimp
2026.03 GitHub ⭐ …
CrabShrimp is an open-source multi-agent runtime framework for collaborative decision-making and autonomous evolution. It integrates Dragon-King orchestration for task planning, Coral-Meeting for consensus-based coordination among agents, and Shell-Molting for post-task self-evolution driven by performance signals. With built-in roles (Planner, Executor, Critic, Verifier, Summarizer), SQLite-backed memory, skill extraction, and resource governance, CrabShrimp enables agent teams that adapt their composition over time and support optional human oversight — compatible with multiple LLM backends via LiteLLM.
TabClaw
2026.03 GitHub Website Platform ⭐ …
TabClaw is an open-source agentic framework that empowers LLMs to reason over complex, real-world tabular data. It decomposes table-centric tasks into structured sub-goals, equips agents with code execution, schema-aware lookup, and formula tools, and coordinates them through multi-step decision workflows. TabClaw is designed to tackle challenges beyond flat QA — including multi-hop joins, conditional aggregation, and cross-table inference — making it suitable for enterprise data analysis and scientific table understanding.
Claw-R1
2026.03 GitHub Website ⭐ …
Claw-R1 is an open-source framework for training reasoning-intensive, tool-using LLM agents via reinforcement learning. It extends standard RL environments with agentic action spaces — including tool invocation, multi-turn interaction, and environment feedback — and supports GRPO-based policy optimization with customizable reward signals. Designed for reproducibility and extensibility, Claw-R1 enables researchers to study how RL shapes emergent reasoning behaviors, tool-use strategies, and decision robustness in large language models.
Mind2Report
2025.12 GitHub Website ⭐ …
Mind2Report is a deep research agent that autonomously transforms open-ended research queries into structured, expert-level reports. It orchestrates multi-round web search and retrieval, synthesizes evidence across hundreds of sources, and produces citation-grounded, section-organized outputs aligned with professional report standards. Targeting domains such as market intelligence, technology landscape analysis, and strategic decision support, Mind2Report bridges the gap between raw web knowledge and actionable analytical deliverables.
Science-Star
2025.07 GitHub Website ⭐ …
Science-Star is an open-source framework for building scientific AI agents capable of tackling graduate-level research tasks. It integrates ReAct-style hierarchical planning, domain-specific tool orchestration (literature search, code execution, symbolic reasoning), episodic memory and self-reflection for iterative improvement, and native evaluation support for the HLE benchmark. Science-Star aims to push the frontier of AI-driven scientific discovery by enabling agents that reason with depth, adapt from feedback, and produce verifiable conclusions.
Agent-R1
2025.04 GitHub Website ⭐ …
Agent-R1 is a large-model agent training framework for end-to-end reinforcement learning fine-tuning. It supports GRPO-based policy optimization with reward signals derived from environment feedback, and equips agents with multi-tool orchestration, persistent long-term memory, and reflective self-correction across interaction turns. Grounded in the DeepSeek-R1 paradigm, Agent-R1 enables researchers to train LLM agents that not only reason under uncertainty but also learn to use tools strategically and recover from failures autonomously.

Benchmarks & Datasets

E-Commerce Search · Recall to Relevance
KuaiSearch Search-based Recommendation Paper
2026.02 GitHub ⭐ …
KuaiSearch: A Large-Scale E-Commerce Search Dataset for Recall, Ranking, and Relevance is built from real user search interactions on Kuaishou. It preserves authentic user queries and natural-language product texts, covers cold-start users and long-tail products, and spans the three key stages of modern search systems: recall, ranking, and relevance judgment.
Scientific Literature · Agentic Evaluation
PaperArena Scientific Literature Mining Paper
2025.10 GitHub ⭐ …
PaperArena: An Evaluation Benchmark for Tool-Augmented Agentic Reasoning on Scientific Literature benchmarks agentic systems on scientific reading and reasoning with tool use. It targets literature understanding, tool-augmented reasoning, and evidence-grounded evaluation over scientific papers and related scholarly workflows.
AI for Science · Chemical Tables
ChemTable Scientific Literature Mining Paper
2025.06 GitHub ⭐ …
Benchmarking Multimodal LLMs on Recognition and Understanding over Chemical Tables introduces ChemTable, a real-world benchmark curated from chemical literature. It supports two core tasks: table recognition and table understanding, with expert annotations over cell polygons, logical layouts, and chemistry-specific semantic labels such as reagents, catalysts, yields, and graphical components.
RAG Evaluation · Dynamic Benchmark
HoH RAG Paper
2025.06 GitHub ⭐ …
HoH: A Dynamic Benchmark for Evaluating the Impact of Outdated Information on RAG studies how retrieval-augmented generation systems fail when knowledge becomes stale. It provides a dynamic evaluation setting for measuring temporal robustness, outdated-information sensitivity, and the impact of knowledge freshness in modern RAG pipelines.
